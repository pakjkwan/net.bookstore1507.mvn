AJAX ( Asynchronous Javascript And XML )

AJAX는 사실 새로운 프로그래밍 언어가 아니라 기존에 있던 위 기술들을 현명하게 사용하는
방법론적 기술이라고 할 수 있다.

기존의 있던 기술들이라 함은 다음을 말한다.
HTML (컨텐츠)
CSS (스타일)
JavaScript, DOM (동적인 출력과 상호작용)
XML (데이터 전송 형식)
XMLHttpRequest (비동기적 데이터 교환)

AJAX의 필요성
기존의 웹 브라우저는 서버로부터 페이지 단위로만 받을 수 있었다.
때문에 간단한 상호작용을 하려고 해도 페이지 전체를 로딩해야했고
이것은 대역폭 낭비를 초래했다.
하지만 AJAX를 사용하면 XML 파일 조각 단위로 데이터를 받아올 수 있기 때문에
전체 페이지를 다시 로드하지 않아도 페이지 일부를 동적으로 갱신할 수 있다.
